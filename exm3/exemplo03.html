Exemplo 03 - Git Basico ad hoc

O git é uma ferramenta de controle de versão de código. É muito importante que um programador domine as modificações que ele faz em projeto de software.

Este documento lista os comandos básicos do git (http://git-scm-com).

Iniciando um repositório

Iniciar um repositório é observar por mudanças ao longo do tempo. Para iniciar o repositório dê o comando:

$ git init
Initialized empty Git repository in /home/igor/teste/.git/

Marcando modificações

As modificações realizadas devem ser marcadas para submissão:

$ git add arquivo.txt

Eviando uma submissão

Para gerar uma submissão, deve-se realizar um commit:

$ git commit -m "Adiciona arquivo"

Verificando o estado das modificações

Você pode verificar o estado das modificações do repositório a qualquer momento:

$ git status
